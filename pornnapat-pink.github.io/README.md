# Pornnapat Pink Portfolio

This is my personal portfolio website, built with HTML, CSS, and JavaScript.

## Features
- **Modern Dark Theme:** Default dark mode with an option to toggle to light mode.
- **Responsive Design:** Optimized for both mobile and desktop screens.
- **JetBrains Mono:** Using JetBrains Mono for a clean, developer-focused aesthetic.

## How to View
The site is hosted on GitHub Pages. You can view the live version at:
[https://pornnapat-pink.github.io/](https://pornnapat-pink.github.io/)

## Technology Stack
- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript
- Google Fonts (JetBrains Mono)
