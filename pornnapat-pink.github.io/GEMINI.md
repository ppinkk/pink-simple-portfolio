# Project: Pornnapat Pink Portfolio

## Project Overview
This is a personal portfolio website developed for Pornnapat Pink. It is a static website built using HTML5, CSS3 (with Flexbox and Grid layouts), and basic JavaScript for interactive functionality. The design emphasizes a modern, minimalist dark theme with JetBrains Mono typography.

## Purpose
The website serves as a showcase for personal projects and contact information, optimized for both desktop and mobile viewing. It is designed to be hosted on GitHub Pages.

## Key Files
- `index.html`: The main structure of the portfolio, including hero, projects, and contact sections.
- `style.css`: Contains all styling, including dark/light theme definitions, responsive grid layouts, and typography.
- `script.js`: Provides interactivity, specifically the theme toggle functionality.
- `README.md`: Project documentation.

## Deployment
The site is configured for deployment on GitHub Pages. Changes are pushed to the `main` branch, which is set to trigger automatic updates on the GitHub Pages platform.

## Development Conventions
- **Themeing:** Use CSS variables (`--bg-color`, `--text-color`, `--accent-color`) for theme management.
- **Typography:** JetBrains Mono is the standard font.
- **Responsiveness:** Use CSS Grid and Media Queries to ensure layout adaptability.
- **Interactivity:** Keep JavaScript minimal, focusing on UI-specific enhancements like theme toggling.
