# Portfolio Website Development Plan - Pornnapat Pink

## Objective
Build a modern, dark-themed portfolio website hosted on GitHub Pages, featuring a hero section, 3 project showcases, and contact information, using the JetBrains Mono font.

## Scope & Impact
*   **Hosting:** GitHub Pages (`pornnapat-pink.github.io`).
*   **Design:** Modern, minimalist dark theme.
*   **Typography:** JetBrains Mono.
*   **Structure:**
    *   **Hero:** "Pornnapat Pink"
    *   **Projects:** 3-project showcase.
    *   **Contact:** Socials/Email section.

## Implementation Steps

### Phase 1: Project Setup
1.  Initialize local folder `pornnapat-pink.github.io`.
2.  Create `index.html`, `style.css`, `script.js`.
3.  Set up local git repository.

### Phase 2: Core Development
1.  **HTML Structure:**
    *   `index.html`: Header, Main (Hero + Projects), Footer (Contact).
2.  **CSS Styling (`style.css`):**
    *   Import JetBrains Mono from Google Fonts.
    *   Define CSS variables for dark color palette.
    *   Implement Flexbox/Grid for layout.
    *   Add responsive design media queries.
3.  **JavaScript (`script.js`):**
    *   Add basic interactivity (e.g., smooth scroll, hover effects).

### Phase 3: Deployment
1.  Push code to GitHub repository `pornnapat-pink.github.io`.
2.  Enable "GitHub Pages" in repository settings.

## Verification
*   Confirm layout responsiveness on mobile/desktop.
*   Verify all links (socials, project links) are functional.
*   Ensure site is live at `https://pornnapat-pink.github.io/`.
