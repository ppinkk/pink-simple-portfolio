# Project: Pink Simple Portfolio

## Project Structure
- `docs/`: Contains the portfolio website (HTML, CSS, JS). This folder is used for GitHub Pages deployment.
- `plan/`: Project planning and session logs.
- `GEMINI.md`: This file (Project guidelines).
- `guide-deploy-statichtml.md`: Reference guide for deployment.

## Git Workflow
- **Branch:** `main` (Default).
- **Remote:** `https://github.com/ppinkk/pink-simple-portfolio.git`.

### Standard Procedure for Changes
1. Make changes to files.
2. Stage changes: `git add .`
3. Commit changes: `git commit -m "Describe your changes"`
4. Push to GitHub: `git push origin main`

## Deployment
- Hosted on GitHub Pages.
- Source: `main` branch, `/docs` folder.
- Live URL: `https://ppinkk.github.io/pink-simple-portfolio/`
